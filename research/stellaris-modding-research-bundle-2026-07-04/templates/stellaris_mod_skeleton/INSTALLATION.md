# Installing the skeleton locally

1. Copy `my_codex_mod.mod` into your Stellaris user mod directory.
2. Copy the `my_codex_mod/` folder into the same directory.
3. Edit `my_codex_mod.mod` and set the `path` value to the absolute path of the copied `my_codex_mod` folder.
4. Enable “My Codex Mod” in the Paradox Launcher playset.
5. Launch Stellaris 4.4.x with only this mod enabled.
6. Check `Documents/Paradox Interactive/Stellaris/logs/error.log`.

Windows target folder:

```text
%USERPROFILE%\Documents\Paradox Interactive\Stellaris\mod```

Linux target folder:

```text
~/.local/share/Paradox Interactive/Stellaris/mod/
```

macOS target folder:

```text
~/Documents/Paradox Interactive/Stellaris/mod/
```
