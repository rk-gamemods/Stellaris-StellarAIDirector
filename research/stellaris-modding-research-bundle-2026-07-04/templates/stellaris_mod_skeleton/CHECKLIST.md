# Skeleton validation checklist

- [ ] Outer `.mod` file copied to user mod directory.
- [ ] Inner `descriptor.mod` remains inside `my_codex_mod/`.
- [ ] Outer `path` points to the actual local folder.
- [ ] `supported_version` matches target branch.
- [ ] Localisation file remains UTF-8 with BOM.
- [ ] Mod appears in launcher.
- [ ] New game starts with mod enabled.
- [ ] `error.log` checked after start.
- [ ] Example civic replaced/removed before release.
- [ ] on_action key validated against current vanilla before release.
