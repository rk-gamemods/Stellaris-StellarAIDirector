# Examples

These snippets are implementation patterns, not universal drop-in files.

Before using any snippet:

1. Verify the folder supports the shown block in local Stellaris 4.4.4 or CWTools.
2. Resolve the active load-order winner for same-ID objects.
3. Preserve DLC gates, localisation, icons, resources, prerequisites, and parent-mod hooks.
4. Use provenance comments for every full-object override.
5. Treat economic_plan and ai_budget examples as workflow templates until exact local syntax is copied.
