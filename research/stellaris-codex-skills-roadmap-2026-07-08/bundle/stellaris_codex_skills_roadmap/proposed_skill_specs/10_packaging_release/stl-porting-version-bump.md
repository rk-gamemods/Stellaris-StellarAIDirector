# stl-porting-version-bump

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Build a porting checklist from vanilla diff, deprecated keys, patch notes, schema changes, save risk, and regression saves.

## Trigger

Stellaris patch update, beta branch, or version bump.

## Do not use when

No target game-version change.

## Likely source references or evidence types

- patch notes
- vanilla depot diff
- CWTools schema
- logs
- source inventory

## Related skills commonly chained

- `stl-version-vanilla-baseline`
- `stl-vanilla-docs-diff`
- `stl-savegame-safety`

## Priority

must-have

## Scope-control notes

One source/target version pair only.
