# stl-trigger-effect-docs

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Verify exact trigger/effect names, parameter forms, scope expectations, and generated-doc freshness before using them.

## Trigger

A task uses unfamiliar triggers/effects or old examples.

## Do not use when

The exact command is proven in current vanilla examples and the task does not need scope validation.

## Likely source references or evidence types

- generated script documentation
- Stellaris wiki trigger/effect pages
- vanilla examples
- CWTools diagnostics
- error.log

## Related skills commonly chained

- `stl-scope-script-basics`
- `stl-events-validation`
- `stl-scripted-effects`

## Priority

must-have

## Scope-control notes

Only validate requested commands; do not catalog all triggers/effects.
