# stl-cwtools-schema-lookup

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Find the relevant CWTools CWT rule path and summarize allowed shape, scope, and known schema gaps for one folder/object type.

## Trigger

## Do not use when

- current vanilla examples
