# stl-irony-conflict-map

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Use Irony merged/conflict output to identify file/object winners, FIOS/LIOS behavior, and candidate patch targets.

## Trigger

A mod conflict, overwritten object, UI winner, or load-order problem is suspected.

## Do not use when

A schema error in one local file explains the issue.

## Likely source references or evidence types

- Irony conflict output
- merged files
- mod order
- source inventory
- mod pages

## Related skills commonly chained

- `stl-compat-patch-authoring`
- `stl-merge-semantics-research`
- `stl-load-order-rules`

## Priority

must-have

## Scope-control notes

Analysis only; patch authoring is separate.
