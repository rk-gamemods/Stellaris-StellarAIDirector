# stl-version-vanilla-baseline

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Determine target Stellaris version, vanilla snapshot, DLC/beta assumptions, and whether a vanilla diff is required.

## Trigger

The task mentions 4.x, patch porting, beta/stable mismatch, vanilla overrides, or compatibility with a live stack.

## Do not use when

The exact target version and vanilla snapshot are already provided and not relevant to the edit.

## Likely source references or evidence types

- local game install
- Steam/Paradox patch notes
- Steam depot snapshot
- source inventory
- descriptor supported_version
