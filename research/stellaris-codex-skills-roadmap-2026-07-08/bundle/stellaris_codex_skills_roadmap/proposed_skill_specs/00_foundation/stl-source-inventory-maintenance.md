# stl-source-inventory-maintenance

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Keep source lists, retrieval dates, confidence, caveats, and stale/dynamic-source warnings current for Codex handoffs.

## Trigger

Adding new research references, mod pages, schema links, or evidence files to the project.

## Do not use when

Answering a direct implementation question where sources are already fresh and recorded.

## Likely source references or evidence types

- source_inventory.csv
- source_index.json
- mod_maintainer_guidance_matrix.csv
- Steam Workshop pages
- GitHub releases
- patch notes
