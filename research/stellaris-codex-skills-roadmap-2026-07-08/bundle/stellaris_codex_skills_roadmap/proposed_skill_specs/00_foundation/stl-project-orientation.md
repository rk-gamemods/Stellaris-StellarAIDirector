# stl-project-orientation

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Identify the target mod, repo layout, task type, deliverable, target Stellaris version, and evidence requirements before loading topic skills.

## Trigger

Any nontrivial Stellaris modding task starts, especially when files, compatibility, or validation may be involved.

## Do not use when

A one-line syntax answer or narrowly scoped localization typo is requested.

## Likely source references or evidence types

- repo tree
- README/changelog
- active branch
- local mod descriptor
- project notes
- user-stated target version

## Related skills commonly chained

- `stl-file-layout-descriptors`
- `stl-version-vanilla-baseline`
- `stl-source-triage`

## Priority

must-have

## Scope-control notes

Only classify and route; do not implement gameplay, validate schemas, or perform compatibility analysis.
