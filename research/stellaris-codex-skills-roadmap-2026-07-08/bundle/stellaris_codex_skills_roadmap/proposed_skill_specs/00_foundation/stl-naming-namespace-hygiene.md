# stl-naming-namespace-hygiene

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Define safe naming conventions for object IDs, event namespaces, localization keys, flags, variables, and generated files.

## Trigger

Creating new objects, events, loc keys, scripted values, flags, variables, or generated patch files.

## Do not use when

Only editing existing vanilla/mod IDs without introducing new keys.

## Likely source references or evidence types

- vanilla naming patterns
- existing mod prefix conventions
- event namespace examples
- localization keys
- error logs
