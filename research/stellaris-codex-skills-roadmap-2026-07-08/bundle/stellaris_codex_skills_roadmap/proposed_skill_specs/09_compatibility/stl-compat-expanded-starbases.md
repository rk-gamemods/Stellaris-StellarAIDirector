# stl-compat-expanded-starbases

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Confirm exact workshop identity, then check starbase ship size, levels, modules, UI slots, UMP/UIOD dependencies, and NSC incompatibilities.

## Trigger

Starbase Extended-style mod is active or starbase files are touched.

## Do not use when

Vanilla-only starbase module tweak.

## Likely source references or evidence types

- Expanded Starbases files/pages
- mod_maintainer_guidance_matrix.csv
- Irony conflicts
