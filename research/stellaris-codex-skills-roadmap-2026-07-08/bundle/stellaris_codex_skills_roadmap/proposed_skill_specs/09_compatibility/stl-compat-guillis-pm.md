# stl-compat-guillis-pm

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Check planet modifiers, deposits, events, PD/Gigas/Real Space interactions, and planet generation hooks for Guilli-style planet modifier mods.

## Trigger

Guilli’s Planet Modifiers or a similar planet-modifier mod is active with planet/zone/deposit changes.

## Do not use when

No planet modifier/deposit/event interaction.

## Likely source references or evidence types

- PD/Gigas/Real Space notes
- deposits
- planet modifiers
- runtime tests
