# stl-compat-nsc3

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Check NSC3 ship sizes, section files, starbase interactions, bioships, global designs, UIOD dependency, and save warnings.

## Trigger

NSC3 is active or ship/starbase files are touched.

## Do not use when

Vanilla-only ship mod with no NSC3 active.

## Likely source references or evidence types

- NSC3 files/pages/releases
- Irony merged ship files
- ship designer smoke test
- error.log

## Related skills commonly chained

- `stl-ship-sizes-sections`
- `stl-compat-ship-stack`
- `stl-ship-graph-validation`

must-have

NSC3-specific deltas only; no generic ship overhaul guide.
