# stl-compat-universal-patches

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Check Universal Resource Patch and Universal Modifier Patch interactions, resource topbar display, economy categories, and dependencies.

## Trigger

New resources/modifiers, URP, UMP, or Expanded Starbases-style dependency is present.

## Do not use when

Vanilla-only resources and no universal patches.

## Likely source references or evidence types

- URP/UMP files/pages
- resource_groups
- economic categories
- Irony conflicts
- runtime UI

## Related skills commonly chained

- `stl-resources-modifiers`
- `stl-resource-topbar-ui`
- `stl-compat-expanded-starbases`

## Priority

must-have

## Scope-control notes

Universal patch surfaces only; do not redesign resource economy.
