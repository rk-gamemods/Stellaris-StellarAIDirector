# stl-icons-sprites-gfx

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Add spriteTypes, icons, DDS/TGA assets, atlas paths, and icon references for resources, components, techs, and UI.

## Trigger

Missing icons, new resources/components/tech icons, or spriteType errors.

## Do not use when

Panel layout or localization is wrong.

## Likely source references or evidence types

- vanilla .gfx
- asset paths
- image dimensions
- error.log
- UI screenshots

## Related skills commonly chained

- `stl-interface-gui`
- `stl-localization`
- `stl-asset-pipeline-dds`

## Priority

should-have

## Scope-control notes

2D sprite/icon references only; no portraits or 3D entities.
