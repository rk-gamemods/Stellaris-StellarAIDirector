# stl-asset-pipeline-dds

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Check image formats, sizes, compression, mipmaps, alpha, and predictable asset export conventions.

## Trigger

Creating or fixing DDS icons, event pictures, portraits, flags, or UI images.

## Do not use when

The issue is a missing spriteType path, not image encoding.

## Likely source references or evidence types

- asset source files
- DDS export settings
- vanilla image dimensions
- visual inspection
- error.log

## Related skills commonly chained

- `stl-portraits-species-assets`

later
