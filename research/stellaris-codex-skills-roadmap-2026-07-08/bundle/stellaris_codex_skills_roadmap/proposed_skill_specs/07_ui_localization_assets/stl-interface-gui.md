# stl-interface-gui

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Edit or validate .gui/.gfx layout files, panels, topbar, planet view, ship designer, outliner, and compatibility patch ordering.

## Trigger

UI breaks, new resources are not visible, or planet/starbase/ship UI files are edited.

## Do not use when

Only localization text or icons are missing.

## Likely source references or evidence types

- vanilla interface files
- UIOD files/pages
- Irony conflicts
- screenshots
- error.log

## Related skills commonly chained

- `stl-compat-uiod`
- `stl-resource-topbar-ui`
- `stl-icons-sprites-gfx`

## Priority

must-have

## Scope-control notes

Layout only; delegate resource semantics and icons.
