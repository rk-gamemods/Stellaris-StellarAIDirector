# stl-localization

> Roadmap card only. This is not a finished Codex skill and is not a `SKILL.md` body.

## Purpose

Add, validate, and troubleshoot localization yml keys, language headers, encoding, missing loc, duplicate keys, and dynamic loc references.

## Trigger

Any player-facing text, event text, UI label, tech name, tooltip, or missing localization appears.

## Do not use when

No text or localization keys change.

## Likely source references or evidence types

- vanilla localization
- localization modding docs
- CWTools loc diagnostics
- error.log
- localization logs

## Related skills commonly chained

- `stl-events-validation`
- `stl-packaging-descriptor-release`
- `stl-naming-namespace-hygiene`

must-have

## Scope-control notes

Text keys only; no UI layout or icon work.
